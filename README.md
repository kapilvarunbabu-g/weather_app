# Weather App 🌦️

A simple Python weather application using OpenWeatherMap API.

## Features
- Get weather by city name
- Shows temperature
- Shows humidity
- Shows wind speed
- Shows weather condition

## Technologies Used
- Python
- Requests Library
- OpenWeatherMap API

## How to Run

1. Install requests library

pip install requests

2. Run the program

python weather.py

## API Used
https://openweathermap.org/api
